# png2chp16

by Pirota Pirozou

for X68000

## 最初に

このツールはpng画像をX68000の16x16px 256色グラフィックベタデータ形式に変換するツールです。主にグラフィック画面にマップチップを描くために使用します。

いわゆるオレオレツールで不特定多数のユーザーが使うようには考慮されてませんが、いちおうPublicにしておきます。

Visual Studioのプロジェクトファイルをはじめ、 mingw64, X68k上のGCCなどでコンパイルできるように、多彩なMakefileを用意しています。

## 使い方

作りかけでまだちゃんと動作しません。
